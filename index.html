Favorite Things

Just One Bag

For this project, we need to use knowledge about grid to split the content into 2 or 3 parts in order to list graphs and table in a proper size and position. I met a problem about the padding size of the submit table. Finally, I solved this problem by adding more <section> in the table, so that I can change the property of the table partially.
